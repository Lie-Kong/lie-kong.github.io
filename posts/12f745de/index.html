低价订阅Termius Pro，年付不到20CNY！

Created2025-05-30|Updated2025-06-01|小知识
|Post Views:|Comments:

前言

Windows 上的 SSH 工具很多，像是老牌的 XShell、FinalShell、MobaXterm，或者是新秀 Electerm，功能上它们都非常的完善，但是与 Termius 相比，都少了跨平台与云同步两个特性。尽管有软件能够同时支持 Windows、macOS、Linux，但它们都没有移动端，而 Termius 不仅在PC端通用，在PE端也能通用。
订阅 Pro版 之后，就可以使用云同步了，大大方便了设备的迁移。众所周知，国外软件直接订阅的价格通常是比较昂贵的，Termius 也是如此，官网订阅的价格高达 10 美元每月，令人望而却步。最近听闻土区仅需20CNY/年，我便来实操体验一下。

开始操作！

充值TRY

  1. 更改账号地址
    首先需要前往 Microsoft 账户将国家或地区修改为土耳其(红框中)
  2. 购买礼品卡
    去某宝/某鱼购买xbox土区礼品卡(我买的100TRY,花了20CNY)
  3. 连接你的土耳其节点
  4. 前往礼品卡兑换页面，将购买到的礼品卡填入其中，确保上面显示的是土耳其(红框中)

5.兑换成功后，账号余额会有 100 里拉

订阅Termius

Tip:订阅无需开启加速器，无需修改时区

  1. 将 Windows 设置中的国家或地区修改为土耳其
  2. 前往 Microsoft Store 下载 UWP 版 Termius （官网版无法通过微软商店订阅）
  3. Termius登录账号，点击setting升级订阅，点击后在打开的订阅页面中，可以看到已经显示土耳其区的价格了
  4. 支付前需要填写个人资料地址，通过地址生成器生成填入
  5. 使用刚刚兑换的 100 里拉余额进行支付即可完成订阅

支付完成即可开始享用

可能遇到的问题与解决方法

  1. 支付的时候,商店的小窗口闪几下就不见了
    解决方法：开启UAC(用户账户控制)即可
    开启UAC步骤如下：
    • 打开控制面板：可以通过在开始菜单中搜索“控制面板”来打开。
    • 选择“用户账户”。
    • 再次选择“用户账户”
    • 点击“更改用户账户控制设置”。
    • 在弹出的窗口中，可以通过拖动滑块来选择不同的通知级别，以开启或关闭UAC。拉到始终通知后，点击“确定”保存更改
  2. 前面很顺利,到最后付款更改store区域!都改了都不行
    解决方法：等几天或直接用土区ip注册一个新号
  3. 深水宝买礼品卡可能碰到黑卡，下场是封号或强制退款
