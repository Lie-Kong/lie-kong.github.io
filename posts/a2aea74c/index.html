解决hexo deploy后Custom domain被重置的问题

Created2025-05-10|Updated2025-07-31|小知识
|Post Views:

前言

我的Blog搭建在GitHub Page上，一开始使用的是Github分配的github.io域名，一次偶然，在itdog上测了下http，有点惨不忍睹，便搜了些教程，挂上了cdn。

起因

几天前，我更新了一下文章，过了一会儿，与我交换友链的朋友叫我修一下Blog，我打开一看好家伙，怎么回事，打开GitHub的setting才发现原来是Custom domain被清除了。

解决方法

  1. 先在Github配置正确的Custom domain
  2. 在hexo的source目录下创建名为CNAME的文件，写入Custom domain
  3. 重新进行hexo三连 hexo clean && hexo g && hexo d
  4. 打开博客，没有问题，再检查一下Github，也是正常的，没有被重置

这样完成后自定义域名便不会被重置。
